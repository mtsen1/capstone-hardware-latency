# Capstone Project: Hardware Confounding in Remote Cognitive Assessments

## Project Overview
This research investigates the impact of hardware-level graphics rendering (software-based vs. hardware-accelerated) on response time (RT) measurements in remote, web-based clinical trials. We demonstrate that rendering-induced visual instability (measured via Frame Coefficient of Variation) significantly biases cognitive assessment metrics, often mimicking cognitive deficits.
